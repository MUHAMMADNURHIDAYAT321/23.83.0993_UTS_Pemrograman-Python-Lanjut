FITUR 1 
 Color Wave Shadow 3D
Animasi bintang bergerak zig-zag dengan warna yang berubah mengikuti gelombang sinus (color wave). Setiap bintang memiliki bayangan 3D berwarna abu yang mengikuti pergerakan.

Fitur 2  Sparkle Effect Warp Mode
Animasi zig-zag dengan distorsi gelombang (warp) sehingga bergerak naik turun secara dinamis. Di sekitar bintang muncul efek percikan cahaya (sparkle) berwarna-warni secara acak.
